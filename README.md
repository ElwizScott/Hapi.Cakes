# Hapi.Cakes – Custom Cake Ordering Website

A full-stack web application for showcasing custom cake designs and handling customer inquiries.

This project is built as a real-world portfolio project using modern backend and frontend technologies.

---

## Tech Stack

### Frontend
- React
- Axios
- TailwindCSS
- React Router

### Backend
- Spring Boot
- MongoDB
- Spring Security (JWT)
- Java Mail Sender

---

## Project Structure

cakehouse/  
├── frontend/ # React application  
└── backend/ # Spring Boot REST API  

---

## Features
- Cake gallery (previous works)
- Custom cake order form
- Image upload for design inspiration
- Email notification for new orders
- Admin management dashboard
- Chat integration (WhatsApp / Live Chat)
